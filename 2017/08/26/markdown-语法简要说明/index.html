链接

外部链接

文档中的文字链接外部网站的 + 样式：[输入文字](网址)

效果：

这是我的网站：gazelle 的链接

但是这样书写会造成阅读的不方便，所以可以使用另外一种样式。

  • 样式：[输入文字]
  • 样式：[输入文字]: 网址 （可以放在文档中其他任意地方，例如文档中或者结尾。并且链接网址不会出现在文档中。）

效果:

点击名称访同网站 ：Google ,Yahoo ,MSN。

内部链接

在文档中，跳转到其他位置。

  • 样式：[输入文字](# 标号)

脚注

脚注功能需要Pandoc的支持。

  • 样式：输入的文字[^1]输入的文字
  • 样式：[^1]: 我是脚注

效果：

输入的文字1输入的文字
